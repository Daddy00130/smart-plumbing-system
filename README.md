# Smart Plumbing System
A PHP & MySQL web application for managing plumbing services, customers, and technicians.

## Features
* *Admin Dashboard:* Manage service requests and plumbers.
* *Customer Portal:* Request plumbing services online.
* *Plumber Dashboard:* View assigned jobs and update availability.

## How to Install
1. Clone this repository.
2. Import the smart_plumbing_db.sql file into your phpMyAdmin.
3. Place the folder in your htdocs directory.
4. Open localhost/smart-plumbing-system in your browser.
